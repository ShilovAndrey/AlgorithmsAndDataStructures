Создать массив, содержащий объекты класса Person и обладающий рядом методов

Класс Person должен обладать:

- переменными фамиля, имя, возраст

- методом displayPerson(), выводящим информацию об объекте
- методом getLast(), возвращающим фамилию

Класс ClassDataArray должен обладать:

- приватной переменной (массивом) типа Person
- приватной переменной, указывающей на количество элементов в массиве

- конструктором с параметром типа int maxSize
- методом возвращающим размер массива(количество элементов)
- методом insert(String last, String first, int age) для добавления объекта Person в массив	
- методом find, реализующим линейный поиск и возвращающим найденный элемент
- методом delete
- методом display
- методом insertionSort

- найти значение ключа

- удалить 3 элемента
