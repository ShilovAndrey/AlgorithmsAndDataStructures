
Реализовать Stack на базе LinkList.
Пользователь класса вызывает push() и pop() для вставки и удаления элементов.

Метод main():
- создает объект стека
- заносит в него два элемента
- выводит содержимое стека
- заносит еще два элемента
- выводит содержимое стека
Результат:
Stack (top-->bottom): 40 20
Stack (top-->bottom): 80 60 40 20
Stack (top-->bottom): 40 20
