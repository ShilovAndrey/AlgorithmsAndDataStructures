Программа разбора скобок BracketsApp.

Пользователь вводит сообщение, программа обрабатывает символы скобок. Если скобка открывающая, то заносит в стек, если закрывающая, то выводит из стека верхний элемент и сравнивает. При совпадении выводится ошибка.

Поиск парных скобок выполняется методом check() класс BracketChecker
- для хранения данных используется стек (класс StackZ)
- класс BracketChecker в конструкторе принимает строку и обрабатывает ее в методе check()
- метод check() использует класс StackZ.
- метод main() класс BracketsApp в цикле запрашивает у пользователя строку текста, создает объект класса BracketChecker, передавая полученную строку в аргументе, после чего вызывает метод check() объекта BracketChecker. Если в ходе разбора обнаружены ошибки, метод check() выводит ошибки о них, в противном случае синтаксис парных скобок провален
- метод check() по возможности сообщает номер символа, в котором была обнаружена ошибка и обнаруженный неправильный символ
