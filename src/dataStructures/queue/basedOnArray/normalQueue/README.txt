
Реализовать очередь в виде класса Queue. Класс содержит типичные методы
insert(), remove(), peek(), isEmpty(), isFull() и size().

В методе main() класса QueueApp создать очередь из 5 ячеек, вставить в нее 4 элемента, затем удалить 3 элемента и вставить еще 4. При шестой вставке происходит циклический перенос. Затем извлечь и вывести значения всех элементов. Результат должен выглядеть так:
40 50 60 70 80
