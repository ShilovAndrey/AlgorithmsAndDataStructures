
Создать класс реализующий приоритетную очередь на базе массива, с хранением всех элементов в порядке сортировки.

Очередь упорядочена по возрастанию приоритетов, то есть элемент с наименьшим ключом обладает самым высоким приоритетом, и извлекается при вызове remove().
Элемент с наименьшим значение ключа всегда должен храниться на вершине массива (в ячейке с наибольшим индексом), а элемент с наименьшим значением ключа - в ячейке с индексом 0.
Класс должен реализовывать следующие методы: insert(), remove(), peekMin(), isEmpty(), isFull().

В методе main() класса PriorityApp сначала вставить в очередь 5 элементов в произвольном порядке, затем извлечь и вывести их.
