
Задание:

Создать класс RecOne, который должен содержать:

- переменную ArrayList, инициирующуюся в конструкторе
- метод, возвращающий значение факториала переданного числа
- метод, принимающий количество элементов и заполняющий массив случайными числами;
- метод, выводящий на экран обратный отсчет от переданного значения
(базовое условие - есть i<=1))
- метод, суммирующий все числа списка и возвращающий сумму
- метод, возвращающий наибольшее число списка
