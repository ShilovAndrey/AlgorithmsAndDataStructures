
Класс OrderedArray должен содержать:

- переменную типа long[]
- переменную типа int для количества элементов

- конструктор с параметром типа int size
- методом возвращающим размер массива(количество элементов)
- методом find, реализующим линейный поиск и возвр. булево значение
- методом findBin, реализующим бинарный поиск и возвращающим индекс элемента
- методом insertSort() в отсортированный массива	
- методом delete
- методом display
